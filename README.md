# Netology Pytorch Project

Project Description

Setup Instructions
1. Clone the repository:
    ```bash
    git clone git@github.com:BalshowAndrew/netology-pytorch-project.git
    cd netology-pytorch-project
    ```

    
2. Create a project:
    ```bash
    uv python pin 3.11
    uv init
    uv venv
    uv add --requirements requirements.txt
    ```
